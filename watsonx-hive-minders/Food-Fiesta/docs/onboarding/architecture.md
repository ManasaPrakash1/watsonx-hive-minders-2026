| ⚙️ Services | UserServices, AdminServices ⚠️, ProductServices, OrderServices, Logic.countTotal() (utility) | All annotated @Component (not @Service). AdminServices.update() has a bug: ID not set on entity before save ⚠️. Optional.get() without guard in getAdmin/getUser/getProduct ⚠️ |
| 🗄️ Repositories (CrudRepository) | UserRepository, AdminRepository, ProductRepository, OrderRepository | findAll() returns Iterable<T> — always cast to List<T> explicitly |
| 💾 Database | customer_info ── User, admin ── Admin, product_table ── Product, order_info ── Orders ⚠️ | H2 in-memory (dev), PostgreSQL (Docker/prod). Orders.totalAmmout has intentional double-m typo — never fix ⚠️ |
| 💡 HTTP Session | "loggedInAdmin" → String (email), "loggedInUser" → User entity | Admin session stores a String; user session stores a full JPA entity |

## Flux de requête

`Browser` → `Spring Security (route guard)` → `Controller (session check)` → `Service (@Component)` → `Repository (CrudRepository)` → `H2 / PostgreSQL` → `Thymeleaf view` → `Browser`
